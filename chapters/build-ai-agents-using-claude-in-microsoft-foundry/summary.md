# Summary: Build AI agents using Claude in Microsoft Foundry

Chapter slug: `build-ai-agents-using-claude-in-microsoft-foundry`

## Abstract

This workshop demonstrates building AI agents using Claude models deployed in Microsoft Foundry, a unified platform for enterprise AI application development. The session covers practical implementation of an AI agent for a fictional cupcake shop called Sparkles, utilizing Microsoft Agent Framework and Model Context Protocol (MCP). The workshop begins with deploying Claude models in Foundry's playground environment, then progresses to local development using VS Code. Participants implement an agent capable of customer registration, order processing, and system integration through MCP servers. The demonstration emphasizes enterprise-ready features including security, observability, and governance capabilities built into Foundry. The agent utilizes Claude's reasoning capabilities for multi-step customer interactions, connecting to external systems through over 1,400 built-in connectors. The workshop showcases the transition from prototype to production deployment, highlighting Microsoft Foundry's role in scaling AI agents with enterprise security features like Microsoft Defender, Purview, and EntraID integration.

## Key claims

- Microsoft Foundry provides enterprise-ready AI agent development with built-in security, observability, and governance features
- Claude models excel at planning, code work, and long context understanding, making them ideal for agent applications
- Microsoft Agent Framework enables rapid development of AI agents with minimal code requirements
- Model Context Protocol (MCP) allows AI agents to easily integrate with external systems through standardized interfaces
- Foundry supports over 1,400 built-in connectors for enterprise system integration including SAP and ServiceNow
- The platform enables faster development-to-production cycles compared to building custom agent platforms from scratch
- Claude Sonnet 4.6 serves as an effective model for multi-step reasoning tasks in agent workflows
