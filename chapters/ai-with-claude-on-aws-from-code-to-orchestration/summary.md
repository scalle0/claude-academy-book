# Summary: AI with Claude on AWS: From code to orchestration

Chapter slug: `ai-with-claude-on-aws-from-code-to-orchestration`

## Abstract

This presentation demonstrates deploying Claude Code applications from prototype to production using Amazon Web Services infrastructure. The session covers AWS-Anthropic partnership benefits including multi-billion dollar investment and commitment to 100+ billion in AWS usage. Key technical offerings include three deployment methods: Claude models through Amazon Bedrock with fine-tuning capabilities, Claude Platform in AWS for feature parity with direct Anthropic access, and Claude desktop integration via AWS Marketplace. The presentation emphasizes security features including zero operator access, data sovereignty, private link connectivity, and compliance with regulations like FedRAMP and HIPAA. Technical infrastructure includes Project Rainier AI compute cluster and custom Tranium chipsets optimized for Claude models. The hands-on workshop demonstrates Claude Code integration with Scaledraw tool, covering basic setup through advanced configurations with MCP protocols, sub-agents, custom skills, and hooks. Workshop modules progress from basic AWS Claude Code setup to advanced features including Playwright MCP for automated screenshot workflows, Git integration, and Amazon Bedrock Agent Core for production agent deployment.

## Key claims

- AWS provides the only cloud platform offering Claude model fine-tuning capabilities, specifically for Haiku models
- Amazon Bedrock implements zero operator access ensuring no Amazon or Anthropic personnel can access customer instances
- Project Rainier represents one of the largest AI compute infrastructures built specifically for Claude model training and hosting
- AWS offers three distinct deployment methods for Claude: direct Bedrock integration, Claude Platform gateway, and desktop marketplace solutions
- Custom Tranium chipsets provide optimized performance and cost-effectiveness for Claude token processing
- AWS provides infinite scalability with regional deployment flexibility for GDPR and compliance requirements
- Amazon Bedrock Agent Core maintains full compatibility with open frameworks including Claude Agent SDK
