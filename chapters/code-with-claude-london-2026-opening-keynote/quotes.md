# Notable quotes: Code with Claude London 2026: Opening Keynote

Quotes selected by Claude from the transcript.

> What's happening now is that distance is collapsing again. You describe a problem. And the program shows up. It's the calculator feeling, except the calculator can write a distributed system.

*Captures the core thesis about AI returning programming to its accessible roots while dramatically expanding capabilities.*

> Then last month, Mythos read the entire open BSD source tree. And it found a 27-year-old vulnerability. That survived every human reviewer, every fuzzer, every static analyzer that was thrown at it for almost three decades.

*Demonstrates a concrete breakthrough showing AI capabilities exceeding traditional security tools and human review.*

> Year over year, API volume is up nearly 60%. volume is up nearly 17x on the Claude platform. And on Claude Code, the average developer is now spending over 20 hours a week running Claude.

*Provides key adoption metrics showing explosive growth in developer usage of AI coding tools.*

> As models get smarter, the scaffolding that used to help can hold Claude back. Claude is intelligent and resourceful, and more intelligent models can often get further with generalised primitives.

*Important architectural guidance about how to design systems as AI capabilities improve.*

> Instead of asking Claude to write a project update, you would ask Claude, keep the project on track this week. Instead of asking Claude to produce a financial forecast, Claude would own and update the forecast to keep it accurate over time.

*Illustrates the shift from task-based to outcome-based AI delegation and continuous autonomous operation.*

> One metric I look to to make sense to all of this change is task horizon. So how long can a model work before losing the thread? Last year at this time, models could reliably work for minutes. And today, most users have agents that run for hours.

*Defines a key metric for measuring AI progress and shows dramatic improvement in sustained autonomous operation.*

> EVE Legal, for example, used the advisor strategy, and they told us they got frontier model quality at five times lower cost.

*Provides concrete customer validation of cost optimization strategies using model hierarchies.*

> At Anthropic, we've seen that this has driven a 200% increase in the number of PRs per engineer. Even as our engineering org has scaled substantially.

*Shows measurable productivity gains from AI adoption at a leading AI company.*

> The default isn't I'm going to prompt Claude Code. The default is now I'm going to have Claude prompt Claude Code.

*Captures the fundamental shift toward AI systems that operate autonomously rather than requiring constant human direction.*

> Most people are never going to call the Claude API. Most people are never going to run Claude in a terminal. They're going to experience AI through something one of you built on the Claude platform.

*Emphasizes the critical role of developers as intermediaries who shape how AI reaches end users.*
