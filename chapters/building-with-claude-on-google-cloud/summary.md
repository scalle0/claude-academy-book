# Summary: Building with Claude on Google Cloud

Chapter slug: `building-with-claude-on-google-cloud`

## Abstract

This talk demonstrates how to leverage Claude models hosted on Google Cloud throughout the complete software development lifecycle. The presentation shows how Claude Code, Anthropic's coding agent, can augment traditional development teams by performing tasks typically handled by product managers, UI/UX designers, software engineers, security engineers, and data analysts. The demonstration walks through building and deploying a feedback application using Google Cloud services, showcasing the integration between Claude Code and Google Cloud's developer ecosystem. Key integrations include official Google Cloud skills, the developer knowledge API with associated MCP server, and various MCP servers for BigQuery analytics and Looker dashboards. The talk emphasizes Google Cloud's advantages for running Claude models, including pay-per-token pricing, no message caps, provisioned throughput options, multi-region availability, and seamless authentication through application default credentials. The demonstration covers prototyping with wireframes, building production interfaces, deploying serverless applications on Cloud Run with Cloud Build CI/CD pipelines, security reviews, and analytics dashboard creation using BigQuery and Looker.

## Key claims

- Claude Code can replace or augment multiple roles in software development teams including product managers, UI/UX designers, and software engineers
- Google Cloud provides superior hosting for Claude models with pay-per-token pricing, no message caps, and provisioned throughput options
- The developer knowledge API provides fresh documentation updated every 24 hours that enables Claude to build Google Cloud architectures
- Official Google Cloud skills allow Claude Code to deploy applications directly to Google Cloud services without prior platform knowledge
- MCP servers enable Claude to perform complex analytics in BigQuery and create dashboards in Looker without specialized expertise
- Claude Code's planning mode and subagents can parallelize development tasks to simulate team sprints
- The integration allows building complete CI/CD pipelines using Cloud Build and Cloud Deploy with security reviews and production promotion workflows
